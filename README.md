# SkillBridge — Micro Skill Exchange Hub

SkillBridge is an interactive skill exchange platform where users can offer skills, request help, discover possible matches, earn community exchange points, and export skill exchange records.

This project is built using HTML, CSS, JavaScript, and LocalStorage. It can be deployed directly using GitHub Pages without GitHub Actions.

## Live Demo

Live project: https://danishzulkafli.github.io/skillbridge-micro-skill-exchange/

## Features

- Add skill offers
- Add help requests
- AI-style match suggestions
- Match score calculation
- Category and keyword matching
- Urgency and availability consideration
- Community points system
- Community rank system
- Category breakdown
- AI-style community insight
- Search board items
- Filter offers and requests
- Delete items
- Export data as CSV
- Reset data
- LocalStorage persistence
- Responsive interface
- GitHub Pages deployment without GitHub Actions
